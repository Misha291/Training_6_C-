# Угадай число (C#)

Консольная игра, где программа загадывает число от 1 до 100, а пользователь пытается его угадать.

## Как работает
1. Программа загадывает случайное число от 1 до 100
2. Пользователь вводит варианты
3. Программа подсказывает: «Больше» или «Меньше»
4. После угадывания показывает количество попыток

## Пример
```
Угадайте число от 1 до 100: 50
Меньше
Угадайте число от 1 до 100: 25
Больше
Угадайте число от 1 до 100: 37

Правильно! Вы угадали за 3 попыток.
```

## Запуск
Скомпилируйте в Visual Studio и запустите (Ctrl+F5).
